# 🎮 Projeto: Jogo Multiplayer de Adivinhação por Imagem

### Inspirado no modo “Pop Sauce” do JKLM (apenas como referência estrutural)

---

# 📌 1. Visão Geral do Projeto

Este projeto consiste na criação de um **jogo multiplayer online em tempo real**, jogável no navegador, inspirado no modo **Pop Sauce** do site JKLM.

> ⚠️ Importante: O objetivo NÃO é copiar o JKLM, mas criar uma versão muito mais moderna, imersiva, animada, criativa e com identidade própria.

O jogo será focado em:

* Cultura pop
* Competitividade em tempo real
* Velocidade de resposta
* Experiência visual altamente impactante
* Forte apelo social e potencial viral

---

# 🧠 2. Conceito Central do Jogo

## 🎯 Mecânica Principal

* Uma imagem aparece na tela.
* Os jogadores precisam adivinhar o que é essa imagem.
* Pode ser:

  * 🎬 Filme
  * 🎵 Música
  * 🎤 Artista
  * 📺 Série
  * 📖 Novela
  * 🧑‍🎤 Personagem
  * 🎮 Cultura pop em geral

---

## 👥 Estrutura Multiplayer

* Mínimo: **2 jogadores**
* Máximo: **16 jogadores**
* Sistema de salas com código
* Um jogador pode:

  * Criar sala
  * Entrar em sala via código

---

## ⏱️ Sistema de Pontuação

A pontuação deve ser baseada em:

1. **Velocidade da resposta**
2. **Ordem de acerto**
3. **Precisão**
4. **Streak (sequência de acertos)**
5. **Possíveis combos**

### 🏆 Regras principais:

* Quem responde primeiro → ganha mais pontos
* Quem responde por último → ganha menos pontos
* Quem erra → não pontua naquela rodada
* Pontuação visível em tempo real
* Ranking dinâmico

---

# 🖥️ 4. Telas do Sistema

---

# 4.1 Tela Inicial (Home / Entrada)

## 🎯 Objetivo

Primeiro impacto. Deve impressionar.

## Elementos obrigatórios:

* Logo do jogo (nome criativo e marcante)
* Botão: “Entrar em uma sala”
* Botão: “Criar sala”
* Fundo animado
* Sons sutis de interface

## Estilo:

* Gradiente animado
* Elementos com glow
* Hover exagerado e satisfatório
* Tipografia forte e ousada
* Micro animações

---

# 4.2 Tela de Cadastro do Jogador

Antes de entrar na sala, o jogador deve configurar seu perfil.

## Campos obrigatórios:

* Nome do jogador (obrigatório)
* Foto / Avatar (opcional)

  * Upload de imagem
  * Ou seleção de avatares pré-definidos
* Campo para inserir código da sala
* Botão para criar nova sala

---

## Experiência Visual:

* Card central flutuante
* Efeito de leve tilt 3D no avatar
* Animação ao digitar nome
* Efeito de brilho ao confirmar
* Botão com efeito magnético ao cursor
* Transição fluida para próxima tela

---

# 4.3 Tela de Lobby (Sala)

## Elementos obrigatórios:

* Código da sala destacado
* Lista de jogadores (avatar + nome)
* Indicador de pronto
* Contador de jogadores (ex: 5/16)
* Botão “Iniciar Jogo” (somente host)

---

## Extras criativos:

* Avatares animados
* Reações rápidas (emojis)
* Som ao entrar jogador
* Parallax leve no fundo
* Destaque visual para host

---

# 4.4 Tela Principal do Jogo

## Layout sugerido:

### 🔝 Topo:

* Timer visual estilizado:

  * Barra energética
  * Contador circular animado
  * Efeito de tensão visual crescente

### 🖼️ Centro:

* Imagem grande da rodada
* Transição suave entre imagens

### ⌨️ Parte inferior:

* Campo para digitar resposta
* Feedback instantâneo (acerto/erro)

### 📊 Lateral:

* Ranking ao vivo
* Pontuação dinâmica
* Mudança de posições animada

---

## 🎯 Feedback Visual

### Ao acertar:

* Pontos “explodindo” na tela
* Partículas
* Efeito sonoro satisfatório
* Nome subindo no ranking com impulso animado

### Ao errar:

* Vibração leve
* Feedback visual discreto
* Som sutil

---

# 4.5 Tela de Resultado da Rodada

Mostrar:

* Resposta correta
* Ranking da rodada
* Pontos ganhos
* Destaque do mais rápido

## Visual:

* Transição cinematográfica
* Spotlight no vencedor
* Medalhas animadas
* Efeitos de luz

---

# 4.6 Tela Final (Fim do Jogo)

Mostrar:

* Top 3
* Pódio animado
* Pontuação final
* Botão “Jogar novamente”
* Botão “Nova sala”

## Experiência:

* Confete animado
* Música curta de vitória
* Efeito celebrativo
* Destaque especial para campeão
